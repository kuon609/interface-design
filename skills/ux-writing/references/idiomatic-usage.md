## 実装への適用ノート

### 漢字をひらがなに開く判断基準

漢字をひらがなに「開く」かどうかは、読みやすさとUIの親しみやすさのバランスで判断する。

**ひらがなに開く候補**:

| 漢字表記 | ひらがな表記 | 判断基準 |
|---------|------------|---------|
| 下さい | ください | 補助動詞はひらがなに開く |
| 出来る | できる | 補助動詞はひらがなに開く |
| 致します | いたします | 補助動詞はひらがなに開く |
| 頂く | いただく | 補助動詞はひらがなに開く |
| 等 | など | 副助詞はひらがなに開く |
| 又は | または | 接続詞はひらがなに開く |
| 及び | および | 接続詞はひらがなに開く |
| 但し | ただし | 接続詞はひらがなに開く |
| 予め | あらかじめ | 副詞はひらがなに開くことが多い |
| 殆ど | ほとんど | 副詞はひらがなに開くことが多い |

**漢字のままにする候補**:

| 表記 | 理由 |
|------|------|
| 確認 | 動詞の漢字は原則として漢字のまま |
| 設定 | 動詞の漢字は原則として漢字のまま |
| 削除 | 動詞の漢字は原則として漢字のまま |
| 必要 | 名詞・形容動詞は漢字のまま |

**判断フロー**:

```
その語は補助動詞・副助詞・接続詞・副詞か？
├─ Yes → ひらがなに開く
└─ No → 動詞・名詞・形容詞の本来の意味で使っているか？
    ├─ Yes → 漢字のまま
    └─ No（形式的な用法） → ひらがなに開く
```

**注意**: ひらがなが連続しすぎると逆に読みにくくなる。漢字とひらがなのバランスを意識する。

### 送りがなの原則

内閣告示「送り仮名の付け方」に準拠する。UIテキストでは省略形を避け、送りがなを付ける。

| 省略形（避ける） | 正式表記 |
|---------------|---------|
| 書込 | 書き込み |
| 取消 | 取り消し |
| 申込 | 申し込み |
| 受付 | 受け付け |
| 読込 | 読み込み |
| 繰返し | 繰り返し |

### カタカナ語の長音ルール

語尾のつづりに応じた長音符号（ー）の付け方。

| 語尾 | 長音 | 例 |
|------|------|-----|
| -er | あり | ユーザー、フィルター、マスター、サーバー |
| -or | あり | モニター、エレベーター |
| -ar | あり | カレンダー |
| -y | なし | セキュリティ、アクセシビリティ、プロパティ |
| -ry | なし | カテゴリ、ギャラリ |

**例外**（慣用的に定着したもの）:

| 語 | 表記 | 理由 |
|-----|------|------|
| browser | ブラウザ | 慣用 |
| folder | フォルダ | 慣用 |

**その他のカタカナ表記**:

| ルール | 例 |
|--------|-----|
| 「ウィ/ウェ/ウォ」を使う | ウィンドウ、ウェブ、ソフトウェア |
| 「シミュレーション」が正 | ×シュミレーション |
| 「スマートフォン」を正式表記とする | ×スマホ（略称は避ける） |

### 数字の表記ルール

| ルール | 例 | 注意 |
|--------|-----|------|
| 数えられるものは半角算用数字 | 3件、10名、5個 | 固有名詞・慣用句は漢数字（「一部」「単一」等） |
| 桁区切りは3桁ごとに半角カンマ | 1,000,000円 | |
| 日付（日本語） | 2025/01/01 | |
| 日付（英語） | Jan 01, 2025 | |
| 時刻 | 13:00:00（24時間制） | 秒は省略可 |
| ファイルサイズ | 5GB、10MB | 英字2文字、スペースなし |
| 「か所」「か月」と表記 | 3か所、6か月 | ×箇所、×個所、×ヶ月 |
| 丸数字は使用しない | ①② → 1. 2. | 環境依存文字のため |

### 記号の表記ルール

**全角で表記する記号**:

| 種類 | 記号 |
|------|------|
| 句読点 | 、。 |
| 区切り符 | ・｜：／ |
| 括弧類 | （）「」［］【】 |
| つなぎ符 | ―…〜 |

**括弧の使い分け**:

| 括弧 | 用途 | 例 |
|------|------|-----|
| （ ） | 補足説明 | データ転送（CSV形式） |
| 「 」 | 入力値・強調・引用 | 「1000」と入力してください |
| ［ ］ | ボタン名・設定項目・選択肢 | ［登録する］を押してください |
| 【 】 | ユーザーが命名したオブジェクト | 【2023年上期評価】を削除 |
| { } | 変数（半角） | {会社名}、{ユーザー名} |

**感嘆符・疑問符**:

| 状況 | 表記 | 例 |
|------|------|-----|
| 単独使用 | 全角 | この操作は元に戻せません！ |
| ペア使用 | 半角 | 本当に削除しますか!? |
| 後続文あり | 直後に全角スペース | 削除しますか？　この操作は取り消せません。 |

**句点のルール**:

| UI要素 | 句点 |
|--------|------|
| 見出し | なし |
| 箇条書き（単語のみ） | なし |
| 箇条書き（文章） | あり |
| 説明文 | あり |

**読点（、）とナカグロ（・）の使い分け**:

| 記号 | 用途 | 例 |
|------|------|-----|
| 、（読点） | 順序に意味がある列挙 | 青、黄、赤の順に点灯 |
| ・（ナカグロ） | 順序が任意の列挙 | 青・黄・赤の三色 |

**スペースルール**:

- 数字と英単語の前後に半角スペースを挿入しない
- ✓「PDFファイル」「2020年」
- ✗「PDF ファイル」「2020 年」

**三点リーダー**: 1つで統一（「…」）。紙媒体の慣例（「……」2つ）には従わない。

**コロンによる例示**: 「例：hogehoge」の形式を使う。「（例）hogehoge」「例）hogehoge」は使わない。

### 操作を表す用語の使い分け

UIで操作を説明する際の動詞選択ガイド。

**操作中断の用語**:

| 用語 | 意味 | 使用場面 |
|------|------|---------|
| キャンセル | 未実行の操作を止める | 確認ダイアログの「キャンセル」ボタン |
| 取り消し | 実行済みの操作を止める | Undo操作、承認の取り消し |
| 中断 | 実行中の操作を一時停止する | 長時間処理の一時停止 |

**状態変更の用語**:

| 用語 | 意味 | 使用場面 |
|------|------|---------|
| 解除 | 元の状態に戻す | ロックの解除、フィルターの解除 |
| 差し戻し | ワークフロー内で修正を求めて前段階に戻す | 承認フローでの差し戻し |
| 戻す | タスクを前段階に戻す | 「下書きに戻す」等 |

**UI要素別の操作表現**:

| UI要素 | 推奨する操作表現 |
|--------|----------------|
| ボタン | 「押す」（デバイス横断的） |
| リンク | 「開く」（「押す」との区別） |
| ラジオボタン | 「選択する」 |
| チェックボックス | 「チェックを入れる/外す」 |
| スイッチ | 「有効にする/無効にする」 |
| ドラッグ操作 | 「ドラッグアンドドロップ」 |
| セレクトメニュー | 「選択する」 |

**混同しやすい操作用語**:

| 用語A | 用語B | 使い分け |
|-------|-------|---------|
| 追加 | 登録 | 「追加」が汎用的。「登録」はアカウント等の文脈で使用 |
| 更新 | 再読み込み | 「更新」はデータの編集保存、「再読み込み」は画面のリフレッシュ |
| 複製 | コピー | 「複製」は完成品の生成まで含む。「コピー」はクリップボード操作 |
| アップロード | 取り込む | 「アップロード」はファイル送信、「取り込む」は外部データの適用 |

### 類似用語の選択基準

| 用語ペア | 使い分け |
|---------|---------|
| 「場合」と「とき」 | 条件が複数ある場合、大前提に「場合」、小前提に「とき」を使う |
| 「テンプレート」と「フォーマット」 | ひな型は「テンプレート」で統一 |
| 「デフォルト」と「プリセット」 | 「デフォルト」は初期状態、「プリセット」はシステムが事前設定した状態 |
| 「サンプル」と「ダミー」 | 「ダミー」はネガティブなニュアンスがあるため「サンプル」を使う |
| 「入力必須」と「必須入力」 | 「入力必須」（「入力を必須にする」の語順が自然） |
